# The fix: state lives in committed files

| Need | File |
|---|---|
| Orientation, read first | `MAP.md` — deliverables → backing docs |
| Recurring lookups | quick index in the instructions file |
| Open items across sessions | `TODO.md` — typed: analysis / verify / physical / decision |
| Canonical names | `GLOSSARY.md` — one file owns nomenclature |
| Hard-won environment lessons | gotchas section, *with the story* |

Every change is a commit: who, when, what, **why**. Anything can be
reverted; any two versions can be diffed; collaborators see the same truth.
