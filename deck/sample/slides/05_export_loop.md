# The practice-export loop

![export loop](assets/loop.svg)

Install is **adaptive** (generic → your subject matter); export is
**abstractive** (your lesson → generic form). The manifest records the
mapping both ways, so neither direction relies on memory.

<!-- notes -->
Walk the arrows: vendor, instantiate, improve locally, fold the abstracted
form back, check in upstream by PR. The scrub gate sits on every arrow that
points at the public repo.
