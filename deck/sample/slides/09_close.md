# Getting started

1. Create a repo; open an agent session on it
2. Paste the bootstrap prompt from the README — the agent vendors this
   layer, instantiates the templates with *your* subject matter, and writes
   the manifest
3. Work normally: every session orients from the map, records open items,
   and runs the export gate before merging
