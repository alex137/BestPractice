# The problem: sessions are ephemeral

- Agent sessions (and humans returning after a month) start **cold**
- Chat threads are gone, for practical purposes, when they end
- Opaque memory features can't be inspected, diffed, reviewed, or shared
- The result is a **re-derivation tax** paid every session: re-finding files,
  re-learning environment quirks, re-making settled decisions

**The test both fail:** you can't control what's in context, and you can't
audit what the assistant "knows."

<!-- notes -->
The tax is the hook — everyone who has run multi-session agent work has paid
it. Name it before naming the fix.
